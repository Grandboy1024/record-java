# 环境搭建
1. 理解 JDK(Java Development Kit) 和 JRE(Java Runtime Environment)，并安装 JDK;
2. 什么是环境变量？如何修改环境变量？环境变量的作用？
    1）添加环境变量 JAVA_HOME 
    2）将 JAVA_HOME/bin 添加到环境变量 PATH
    3）使用 echo ${环境变量名} 可打印环境变量的值
3. IntelliJ IDEA：
    1）安装
    2）新建项目
    3）运行项目
    4）出错信息查看
4. VSCode 的使用（非必须）
